# Blinkit Sales Analysis Dashboard (Excel)

## Overview

An interactive Excel dashboard built to analyze Blinkit's grocery sales performance using Pivot Tables, Pivot Charts, Slicers, and KPI cards.

## Business Problem

Analyze sales performance across outlet types, item categories, locations, and customer ratings to identify business insights.

## Business Questions Answered

1. What is the total sales generated by Blinkit?
2. What is the average sales per item?
3. How many items were sold?
4. What is the average customer rating?
5. Which outlet type generates the highest sales?
6. Which item categories contribute the most revenue?
7. Which outlet size performs the best?
8. Which outlet location tier has the highest sales?
9. How does sales performance vary by establishment year?
10. Which product categories have the highest customer ratings?
